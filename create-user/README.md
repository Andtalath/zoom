# zoom-create-user
Create api user in Zoom

### api url
* US = `api.zoom.us`
* EU = `eu01web.zoom.us`

## To run
* Install/Download python: https://www.python.org/downloads/
* Install/Download pip: https://pip.pypa.io/en/stable/installing/
* Run the script: `python create-user.py`
